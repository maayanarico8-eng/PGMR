Group 1 — Philosophical Decisions 
P1 — Fidelity to the Written Memory
The language preserves the memory as expressed by the narrator through the written memory.
The written memory is treated as the sole evidence available to the processor.
The processor's responsibility is not to reconstruct the historical event, infer hidden meaning, or identify an experiential essence beyond what is stated.
Its responsibility is to preserve the memory that the narrator chose to express.
The processor may analyze, organize, and evaluate elements within the written memory in order to identify memory identity and select Representative Words.
However, it may not introduce new semantic content, inferred meanings, symbolic interpretations, emotional explanations, motivations, intentions, cultural assumptions, or contextual knowledge that are not explicitly present in the written memory.
The processor must remain faithful to the memory as written.
All subsequent stages of the methodology operate under this principle.  P1 Clarification — Textual Presence
The written memory remains the sole source of evidence available to the processor.
Evidence includes both explicit lexical content and grammatical structure.
Participants, relationships, recipients, and event roles that are encoded through the grammatical structure of the written memory are considered textually present.
Using grammatically encoded information does not constitute semantic enrichment, because the information originates from the written memory itself.
The processor may use grammatically encoded information.
The processor may not introduce information that is neither explicitly stated nor grammatically encoded in the text.


P1 Clarification — Lexical Form Fidelity
The word label assigned to each Representative Word must be derived from the written memory. The processor may not replace, normalize, or restructure the form used by the narrator with a different lexical form, even if the substituted form is semantically equivalent.

When the written memory uses a specific grammatical form — including inflected pronouns, verb phrases, or noun phrases in specific syntactic positions — that form, or its minimal lexical source within the written memory, serves as the basis for the word label. Substitution is not permitted. If the written memory contains "אותנו", the word label is derived from "אותנו" — not from an uninflected or restructured equivalent such as "אנחנו". If the written memory contains "קורא לי עיתון", the word label reflects that phrase — not a nominalized reconstruction generated by the processor.

The canonicalReferent field may be used to record a normalized or translated description for internal analytical purposes. The word label may not.

This constraint is a consequence of P1: the processor may not introduce lexical forms that were not used by the narrator. Replacing the narrator's own words with processor-generated alternatives — even semantically equivalent ones — constitutes a form of lexical rewriting that P1 prohibits.

P2 — Intended Reader
The pictographic language is designed to be readable by an uninformed viewer.
The viewer is not expected to possess prior knowledge of the narrator, the family, or the original memory.
The goal of the pictographic sequence is not to reproduce the memory verbatim.
The goal is to communicate the central structure and meaning of the memory through a readable visual sequence.
A successful sequence should allow an uninformed viewer to infer the general event, participants, and narrative situation represented by the memory, even if some specificity or nuance is lost.
The language therefore prioritizes interpretability and narrative legibility over private symbolic meaning.  P2 Clarification
Readability should not be interpreted as semantic simplification.
The loss of some specificity or nuance does not permit the loss of semantic categories established by the written memory.
If the memory specifies a grandfather, child, hospital, teacher, learner, or other identity-relevant category, that category should remain recoverable in the visual representation.
The uninformed viewer is not expected to reconstruct the memory verbatim.
However, the visual sequence should preserve the highest level of semantic precision supported by the written memory while remaining readable.

P3 — Semantic Engine
The semantic methodology is recording rather than generative.
Its purpose is to identify, preserve, and represent the memory as expressed by the narrator.
The system does not generate new meaning, symbolic interpretations, emotional explanations, cultural readings, or narrative content.
All semantic content must originate from the written memory. 
P3 Clarification
P3 governs the semantic methodology only.
The restriction against generating semantic content applies to memory analysis, semantic selection, and Representative Word selection.
Questions regarding pictogram realization, visual sourcing, sequence rendering, spatial arrangement, and visual language behavior belong to later stages of the methodology.
These visual stages may employ formal conventions of the pictographic language without introducing new semantic content beyond the written memory.
 
Group 2 — Philosophical Decisions

D1 — Memory Identity
Memory Identity consists of the semantic elements present in the written memory that are necessary for preserving the identity of the memory.
An element belongs to Memory Identity if its removal transforms the memory into a meaningfully different memory rather than the same memory with reduced specificity.
The purpose of Memory Identity is not to preserve every detail of the written memory, nor to reconstruct the historical event. Its purpose is to preserve the memory itself.
The defining question is: "If this element is removed, would the resulting representation still be recognized as the same memory, or would it become a different memory?"
This definition requires a distinction between categorical changes and gradual changes.
Categorical changes alter the identity of the memory itself. They change the participant structure, event structure, relationship, situation, or other aspects of the memory whose preservation is necessary to maintain memory identity.
Gradual changes reduce specificity while preserving identity. They make the memory less precise, less detailed, or less distinctive, but the memory remains fundamentally the same memory.
The boundary between identity-changing elements and detail-level elements must be defined explicitly, because this distinction forms the basis for all later selection decisions.

D2 — Constituent vs Detail
The constituent/detail distinction is derived directly from Memory Identity.
A constituent is an explicitly stated element whose removal would cause the memory to become a meaningfully different memory.
A detail is an explicitly stated element whose removal would reduce specificity, precision, or richness, while preserving the memory's identity.
The distinction is not based on grammatical category, semantic category, event type, participant type, or visual representability.
An action may be a constituent or a detail.
A participant may be a constituent or a detail.
An object may be a constituent or a detail.
A quality, location, language, emotion, or temporal condition may be a constituent or a detail.
The determining question is always: "If this element is removed, does the memory remain recognizably the same memory with less detail, or does it become a meaningfully different memory?"
If the memory becomes a different memory, the element is a constituent.
If the memory remains the same memory with reduced specificity, the element is a detail.
The classification must be determined by its contribution to Memory Identity, not by its semantic category.

D3 — Participant Selection Condition
The narrator should not be selected simply because every memory has a narrator.
The narrator becomes a candidate for selection only when the narrator is explicitly present as a participant within the remembered event.
A participant is any entity whose presence contributes to the structure, relationship, or identity of the memory.
The determining question is not: "Who is telling the memory?”, the determining question is: "Who is participating in the memory?”.
If removing the narrator transforms the memory into a meaningfully different memory, the narrator functions as a participant and must be evaluated like any other participant in the memory.
If the narrator functions only as the author of the memory and not as a participant in the remembered event, narrator representation is not required.

D4 — Representative Words
Representative Words are semantic units selected for visual representation because they have been determined to be necessary for preserving memory identity.

Representative Words are derived directly from Constituents. Every element classified as a Constituent by Stage 1.5 is automatically promoted to Representative Word status. A Constituent cannot be excluded from the Representative Words set without contradicting D1, D2, and the Visual Representation Principle.

No additional identity-based selection step exists between Constituent classification and Representative Word status. The Memory Identity Gate determination produced by Stage 1.5 is the primary basis for membership in the Representative Words set. Elements may additionally enter the Representative Words set through the structural pathway established by R4, when a Must Preserve constraint requires an element that is absent from the standard selection. The basis for inclusion through the structural pathway is recorded in the Consideration Record.

Group 3 — Rules

R1 — Participant Selection
Participant selection does not require an independent rule.
Participant selection is determined by D1 (Memory Identity) and D2 (Constituent vs Detail).
All participants, including the narrator, are evaluated using the same identity test.
The determining question is: "If this participant is removed, does the memory become a meaningfully different memory, or does it remain the same memory with less detail?"
If removing the participant changes memory identity, the participant is a constituent and must be retained.
If removing the participant preserves memory identity, the participant may remain semantic-only.
The distinction between narrator-as-author and narrator-as-participant is therefore not a separate methodological rule. It is a specific application of the Memory Identity framework.

R2 — Action Selection
Actions are not automatically selected because they occur within the memory.
An action should be retained only when it contributes to Memory Identity.
An action is identity-critical when removing it transforms the memory into a meaningfully different memory.
An action is mechanistic when it describes how the memory occurred but its removal preserves the memory's identity.
The determining question is: "If this action is removed, does the memory remain recognizably the same memory, or does it become a different memory?"
Actions that define the remembered event, relationship, or transformation should be retained.
Actions that merely connect, support, or explain other retained elements may remain semantic-only.
The classification of an action must be determined by its contribution to Memory Identity rather than by the fact that it is an action.

R3 — Sequence Position Assignment
Sequence position is not determined by semantic category.
Participants, actions, objects, locations, qualities, and other elements do not occupy fixed positions based on type.

The Event Model is authoritative for sequence assignment. Sequence positions are derived from the chronological structure of the remembered event as reconstructed in the Event Model. The order in which the memory is narrated serves as evidence from which the Event Model is constructed but does not independently determine sequence position.

For the purposes of sequence assignment, narrative structure refers to the chronological unfolding of the event as reconstructed in the Event Model, not the narrative telling order of the written memory.

The pictographic sequence follows the chronological event structure recorded in the Event Model and Semantic Structure. In other words, sequence position is chronological rather than categorical or grammatical. The language preserves the chronological order implied by the remembered event rather than imposing a fixed grammatical hierarchy on semantic categories.

Element-Type Ordering Rules:

Participants — Participants receive sequence positions according to their first chronological appearance in the Event Model. First chronological appearance refers to the Participant's first position in the reconstructed event structure, not its first mention in the written memory.

Locations and Temporal Referents — Locations and Temporal Referents receive sequence positions according to the chronological point at which they enter or contextualize the event structure represented in the Event Model.

Qualities — Qualities derive their sequence position from the element they modify. No independent positioning rule is required.

Relationships — Relationships primarily function as analytical structures that inform the Event Model and Semantic Structure. They are not generally expected to become Representative Words. When a Relationship does survive as a Constituent Representative Word, its sequence expression is mediated through the positions of the elements it connects rather than through an independent sequence position. No independent sequence positioning rule is required for Relationships.

R4 — Must Preserve Consistency Check
D1, D2, and the Memory Identity Gate preserve identity-critical semantic elements.
Must Preserve preserves identity-critical semantic structures and relationships that emerge from the interaction of multiple elements.
These are distinct and complementary methodological functions. Preserving individual elements does not guarantee preservation of the structures and configurations that give those elements their meaning within the memory. A Representative Words set may contain all identity-critical elements while failing to preserve the participant configuration, relational structure, or event structure that defines the memory's identity.
Must Preserve is therefore a logically necessary and independent validation layer, not a contingent implementation safeguard.
Must Preserve is restricted to multi-element structural constraints. Each Must Preserve statement describes a structural requirement whose preservation depends on the combined presence and configuration of two or more elements. Must Preserve entries are derived from two exhaustive sources within the Event Model: Participant Role combinations recorded in Layer 1B, and Relationship entries recorded in Layer 1A. Single-element requirements are governed exclusively by D1, D2, and the Memory Identity Gate and do not generate Must Preserve constraints.
The selected Representative Words set must satisfy both element-level and structure-level preservation requirements before the workflow may continue.
If one or more Must Preserve requirements cannot be satisfied by the selected Representative Words set, the Structural Failure Response defined in Stage 1.7 is applied. A Must Preserve failure first triggers a second Gate evaluation informed by the set-level structural evidence provided by the unsatisfied constraint. If that evaluation produces a Clear Constituent classification, the element is reclassified and promoted through the standard D4 pathway. If the second Gate evaluation still produces a Clear Detail classification, the Must Preserve constraint takes precedence because it evaluates structure-level preservation requirements across the complete selected set. The element is promoted through the structural pathway, and the basis for inclusion is recorded in the Consideration Record.

Visual Representation Principle-
Representative Word selection must remain completely independent from visual representation decisions.
The ability or inability to represent an element visually must not influence whether it is selected.
Representative Words are selected solely according to the semantic methodology.
Once a Representative Word is selected, it must receive a visual representation.
There is no category of "selected but unrepresented" Representative Words.

Clarification — Representative Word ≠ Standalone Pictogram

Selection as a Representative Word commits a semantic element to the representation pipeline. It does not predetermine that the element will receive its own pictogram.

Whether a Representative Word receives standalone pictographic realization, contextual expression through other pictograms, or both is determined exclusively in Rule 2 (Visual Representation Planning). Rule 1 must not exclude an identity-critical element from the Representative Words set merely because Rule 2 may later classify it as Contextual.

Questions regarding pictogram libraries, visual sourcing, pictogram selection, and visual transformation belong to later stages of the methodology and should not influence Stage 1.
The word label assigned to each Representative Word when it is produced by Stage 1.6 is governed by P1 Clarification — Lexical Form Fidelity.
